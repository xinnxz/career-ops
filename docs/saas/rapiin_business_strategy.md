## 1. PRODUCT: Fitur yang Bikin Orang HARUS Beli

### 1.1 Core Features (MVP)

#### A. "CV Roast" — Gratis (Hook Utama)
**Psikologi: Fear + Curiosity**

User upload CV lama mereka, lalu AI "roast" (kritik) CV-nya:
```
┌──────────────────────────────────────────┐
│  📊 Skor CV Kamu: 34/100 (LEMAH)        │
│                                          │
│  ❌ Format tidak ATS-friendly            │
│  ❌ Tidak ada angka/metrik pencapaian    │
│  ❌ Summary terlalu generic              │
│  ❌ Skill section berantakan             │
│  ❌ 78% perusahaan akan SKIP CV ini      │
│                                          │
│  [Lihat CV Versi Upgrade →]              │
└──────────────────────────────────────────┘
```

**Kenapa ini POWERFUL:**
- GRATIS → semua orang mau coba
- Hasilnya MENYAKITKAN → orang jadi takut CV-nya jelek
- Rasa penasaran → "gimana dong yang bagusnya?"
- Natural lead ke produk berbayar

#### B. "Before vs After" Preview
**Psikologi: Contrast Effect + Loss Aversion**

Setelah di-roast, tampilkan CV lama vs CV baru side-by-side:
```
┌─────────────────┐    ┌─────────────────┐
│   CV LAMA 😰    │    │  CV BARU 🔥     │
│                 │    │                 │
│  Times Roman    │    │  Modern Layout  │
│  No structure   │    │  ATS Optimized  │
│  Plain text     │    │  Clean Design   │
│  Boring...      │    │  Professional!  │
│                 │    │                 │
│  Skor: 34/100   │    │  Skor: 92/100   │
└─────────────────┘    └─────────────────┘

         [Download CV Baru — Rp 29.000]
```

**Kenapa ini POWERFUL:**
- Orang MELIHAT perbedaannya secara visual
- CV baru sudah "milik mereka" (nama, data mereka sudah di sana)
- Bayar Rp 29K terasa seperti "menyelamatkan karir", bukan "beli produk"

#### C. "ATS Score Checker"
**Psikologi: Gamification + Fear**

Tampilkan skor ATS (Applicant Tracking System) dari CV mereka:
```
┌──────────────────────────────────────────┐
│  🤖 ATS Compatibility Score             │
│                                          │
│  ████████░░░░░░░░░░░  38%               │
│                                          │
│  Artinya: 62% kemungkinan CV kamu       │
│  LANGSUNG DIBUANG oleh robot HR         │
│  sebelum dibaca manusia.                │
│                                          │
│  CV yang sudah di-upgrade Rapiin:       │
│  █████████████████░░░  94%              │
│                                          │
│  [Upgrade Sekarang — Rp 29.000]         │
└──────────────────────────────────────────┘
```

**Kenapa ini POWERFUL:**
- Angka/skor membuat masalah terasa NYATA
- "Robot HR buang CV kamu" = ketakutan yang valid dan universal
- Solusinya ada di depan mata, tinggal bayar

#### D. Template Pilihan (5-8 Design)
**Psikologi: Autonomy + Ownership**

Kasih user pilihan template:
- **Minimalist** — untuk corporate/bank
- **Creative** — untuk startup/agency
- **Tech** — untuk developer/engineer
- **Fresh Grad** — untuk yang baru lulus
- **Executive** — untuk yang sudah senior

**Kenapa ini penting:**
- Orang suka MEMILIH (merasa in control)
- Template yang berbeda = merasa produknya bernilai lebih

## 2. PRICING PSYCHOLOGY: Strategi Harga

### 2.1 Anchoring (Jangkar Harga)

**JANGAN langsung tunjukkan Rp 29K.** Tunjukkan dulu "nilai sebenarnya":

```
┌──────────────────────────────────────────┐
│                                          │
│  Konsultan CV profesional: Rp 500.000   │
│  Template CV premium: Rp 150.000        │
│  Kelas bikin CV online: Rp 200.000      │
│                                          │
│  cvitae.id: Rp 29.000 (sekali bayar)    │
│                                          │
│  ⚡ HEMAT 94% dibanding konsultan        │
│                                          │
└──────────────────────────────────────────┘
```

**Efek psikologis:** Rp 29K terasa seperti PENCURIAN karena dibandingkan dengan Rp 500K.

### 2.2 Decoy Pricing (Harga Umpan)

Tampilkan 3 paket. Paket tengah adalah yang kamu MAU mereka beli:

```
┌────────────┐  ┌────────────────┐  ┌────────────────┐
│   BASIC    │  │  RECOMMENDED   │  │    PREMIUM     │
│            │  │   ⭐ BEST      │  │                │
│  1 CV      │  │  1 CV          │  │  1 CV          │
│            │  │  Surat Lamaran │  │  Surat Lamaran │
│            │  │  3x Revisi     │  │  Unlimited     │
│            │  │                │  │  Portfolio Web  │
│            │  │                │  │  LinkedIn Opt.  │
│  Rp 29K   │  │  Rp 49K        │  │  Rp 79K        │
│            │  │  (-38% /item)  │  │                │
│ [Pilih]    │  │ [★ Pilih Ini]  │  │  [Pilih]       │
└────────────┘  └────────────────┘  └────────────────┘
```

**Efek psikologis:**
- Basic terasa "kurang"
- Premium terasa "kemahalan"
- RECOMMENDED terasa "pas banget" ← ini yang akan dibeli 60% orang
- Average revenue per user naik dari Rp 29K jadi **~Rp 45K**

### 2.3 Urgency Timer (FOMO)

```
🔥 Harga promo Rp 29.000 berakhir dalam 23:44:12
   Harga normal: Rp 59.000
```

**Efek psikologis:**
- Takut kehilangan (FOMO)
- Memaksa keputusan SEKARANG, bukan "nanti aja"
- Timer ini bisa di-reset per session (banyak SaaS melakukan ini)

## 6. PSYCHOLOGICAL TRIGGERS: Trik Psikologi di Seluruh Journey

### 6.1 Loss Aversion (Takut Rugi)
> Orang 2x lebih termotivasi oleh KETAKUTAN kehilangan daripada harapan mendapatkan.

**Implementasi:**
- "78% CV langsung DIBUANG oleh robot"
- "Setiap hari kamu pakai CV lama = kesempatan yang HILANG"
- "Harga promo BERAKHIR dalam 23 jam"

### 6.2 Social Proof (Bukti Sosial)
> Orang mengikuti apa yang dilakukan orang lain.

**Implementasi:**
- "2,847 orang sudah upgrade CV mereka"
- Counter real-time di landing page: "12 orang baru saja download"
- Testimonial dengan FOTO + NAMA + HASIL nyata
- "Dipercaya oleh alumni UI, ITB, UGM, Binus..."

### 6.3 Reciprocity (Timbal Balik)
> Kalau kamu kasih sesuatu gratis, orang merasa "hutang budi".

**Implementasi:**
- CV Roast GRATIS → user merasa "dapet insight berharga"
- ATS Score GRATIS → user merasa "wah berguna banget"
- Setelah dapat value gratis, bayar Rp 29K terasa seperti "wajar"

### 6.4 Commitment & Consistency
> Setelah orang mengambil langkah kecil, mereka cenderung lanjut.

**Implementasi:**
- Step 1: Upload CV (komitmen kecil, gratis)
- Step 2: Lihat hasil roast (sudah invest waktu)
- Step 3: Pilih template (sudah merasa "ini CV gue")
- Step 4: Bayar Rp 29K (langkah terakhir, tinggal sedikit)

Setiap step membuat mereka SEMAKIN SULIT untuk pergi tanpa beli.

### 6.5 Scarcity (Kelangkaan)
> Sesuatu yang terbatas terasa lebih berharga.

**Implementasi:**
- "Promo Rp 29K hanya untuk 100 orang pertama bulan ini"
- "Template Executive tersisa 23 slot"
- Timer countdown di halaman checkout

### 6.6 Authority (Otoritas)
> Orang percaya pada yang ahli.

**Implementasi:**
- "Dibuat berdasarkan riset dari 10,000+ CV yang berhasil"
- "Template sesuai standar ATS Fortune 500"
- Konten "Tips dari HR" di TikTok = posisi sebagai expert

## 7. GROWTH HACKING: Trik Pertumbuhan Tanpa Budget

### 7.1 Referral Loop
```
Setelah user download CV:

"🎁 Share link ini ke 3 teman, 
 kamu dapat REVISI CV GRATIS (senilai Rp 29K)"

User share → teman klik → teman coba → teman beli
= GRATIS marketing yang terus berputar
```

### 7.2 LinkedIn Viral Loop
Setiap CV yang di-generate punya watermark kecil:
```
"CV dibuat dengan cvitae.id"
```
Ketika user upload CV ke LinkedIn/Jobstreet → brand exposure gratis.

### 7.3 Komunitas & Grup
- Masuk ke grup Telegram/WA "Lowongan Kerja 2026"
- Jangan jualan langsung, kasih tips gratis dulu
- Setelah dipercaya: "btw gue bikin tools buat rapiin CV, link di bio"

### 7.4 Partnership dengan Career Center Kampus
- Hubungi career center kampus-kampus
- Tawarkan akses gratis untuk mahasiswa tingkat akhir
- Mereka promote ke ribuan mahasiswa = user acquisition gratis
- Beberapa akan upgrade ke paket berbayar
